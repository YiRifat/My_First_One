HTML

HTML - HyperText Markup Language. HTML is a markup language that defines the structure of your content. HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to make it appear a certain way, or act a certain way. The enclosing tags can make a word or image hyperlink to somewhere else, can italicize words , can make the font bigger or smaller, and so on. To know more, Click Here

Tags

HTML tags are used to hold the HTML element. HTML element holds the content. HTML attributes are used to describe the characteristic of an HTML element in detail. HTML tag starts with < and ends with > Whatever written within a HTML tag are HTML elements.

Some Basic

For every html coding there are 4 basic tags, which must be used. They are-

  1. Html Tag
  2. Head Tag
  3. Body Tag
  4. Title Tag

First of all we need to apply the html tag. After that we have to put 2 more tags inside of html tag. Those are head tag & body tag. At last the title tag. It has to be placed in head tag. In body tag, we usually write almost eyer tag inside of the body tag.

Usually we use Visual Studio Code for html. If we run exclamatory mark ( ! ) , it will put all 4 basic tags (including some other tags) for you. Basically we can say that it is a shortcut.
